# Packaged-Nutrient-Paste-Meal-Feeder
Nutrient Paste feeder that can select between nutrient pellet(from nutrient pellet mod 1.5 still works in 1.6), nutrient paste meal and Packaged Nutrient Paste


Derived mod from ThatHittman Nutrient - Paste Expanded - Nutrient Pellets
Modder released mod for 1.5 but works fine in 1.6

Adds a third toggle to Vanilla Nutrient Paste Expanded - Nutrient Paste Feeder.
From Nutrient Pellets or Nutrient Paste Meal

The new toggle adds Packaged Nutrient Paste
Defname MealPackagedNutrientPaste

Packaged nutrient paste comes ethanw80 - Packaged Nutrient Paste Forked or you can declare your own packaged nutrient paste using said def name above.

Packaged nutrient paste requires two paste meals from the tank.

Dependencies

Vanilla Nutrient Paste Expanded
Packaged Nutrient Paste Forked(if you will not make your own packaged nutrient meal)
Not compatible with Vanilla Nutrient Paste Expanded: Reimagined Progression
I mean if you are already using that it has nutrient kibble, packaged nutrient paste or more that you may or may not find bloated.
